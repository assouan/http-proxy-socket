# Assouan HTTP Proxy Socket

HTTP and HTTPS proxy socket transports for the A PHP libraries.

```bash
composer require assouan/http-proxy-socket
```

Requires PHP 8.5 or later.

## Classes

- `A\Proxy\HttpProxySocket`
- `A\Proxy\HttpsProxySocket`
- `A\Proxy\ProxyConfig`
- `A\Proxy\ProxyType`
- `A\Proxy\HttpProxyBridge`
